# 💕 性爱AI

日系甜美风纯静态 AI 色情聊天网站。使用 DeepSeek V4 Pro API。

## 功能

- 🔞 8 个预设角色（JK、伪娘、小正太、BDSM、黑市等）
- ✏️ 自定义角色 + 本地存储
- 💬 流式聊天，支持聊天历史持久化
- 👤 用户身份自定义
- 📤 聊天记录导出
- 📱 响应式设计（桌面+手机）

## 部署

1. Fork/Clone 此仓库
2. 在 `js/api.js` 中修改 `API_KEY` 为你的 DeepSeek API Key
3. Push 到 GitHub，启用 GitHub Pages（Settings → Pages → Deploy from branch → main → / (root)）

## 技术栈

纯静态 HTML + CSS + JS，零依赖（仅 Google Fonts CDN）

- DeepSeek API (stream chat)
- localStorage 持久化
- OKLCH 色彩系统
- 响应式布局
